Name:
=====
 freertos_tetris

Description:
============
 Example of a Tetris clone running under FreeRTOS.


This example implements the classic game of Tetris. It monitors On-board buttons, and moves tetraminos in response.
To save power, this application is compiled without print
statements by default. To enable them, add the following project-level
macro definitions.

AM_DEBUG_PRINTF

If enabled, debug messages will be sent over ITM.
